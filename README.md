# Clarity365

Business Central extension project for Clarity365.

## Included Starter Modules
1. Base setup and administration
2. Role Center and profiles
3. Permission sets for user and admin
4. Work Item feature module (table, enum, list/card pages, management codeunit)
5. GitHub Actions CI and manual release-tag workflow

## Getting Started
1. Open this folder in VS Code.
2. Download symbols from the AL extension command palette.
3. Build and publish to your Business Central sandbox.
4. Assign profile `CLARITY` or `CLARITYADMIN` and permission set `CLARITY` or `CLARITYADMIN`.
